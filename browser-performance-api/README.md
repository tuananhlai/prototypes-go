# Browser Performance API - Minimal Demo

## Run

```bash
go run .
```

Open `http://localhost:8080`.

## What it shows

- [Navigation Timing (TTFB, DCL, Load)](https://developer.mozilla.org/en-US/docs/Web/API/Performance_API/Navigation_timing)
- [Paint Timing (FCP)](https://w3c.github.io/paint-timing/)
- [User Timing (Mark + Measure)](https://w3c.github.io/user-timing/)
